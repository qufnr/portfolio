프로젝트
음식 배달 웹사이트
2021.07 ~ 2021.08 (개인 프로젝트)
평소 우리가 사용하는 배달 음식 앱을 참고하여 제작한 음식 배달 웹페이지 기본 틀입니다. 최근에는 코로나 사태로 인해서 많은 사람들이 매장 식사보다는 음식을 배달 시켜 먹는 경우가 증가하였습니다. 이로 인해 배달 앱은 어떤 방식으로 되어있는가 생각하고 참고하여 제작하였습니다.

Vue.js를 사용하여 UI를 제작하고, Back-end 기술(Java, Spring, MySQL)을 사용하였습니다. 처음 사용한 Kakao의 주소 API를 사용하여 사용자가 검색한 주소를 웹페이지에 가져와 사용할 수 있는 기능을 제작했습니다.

해당 페이지를 만들면서 Vue.js에 대한 지식도 많이 얻고, API의 중요성을 알 수 있게 해준 프로젝트였습니다.
주요 기능
음식 주문, 음식 목록, 음식 상세 정보, 배달지 검색, 장바구니

카셰어링 웹사이트
2021.08 ~ 2021.09 (4인 팀 프로젝트)
급히 어디를 가야하거나, 드라이브를 하고싶은 사람을 생각하여 만든 카셰어링 웹페이지 틀입니다. 처음, 학교에서 팀원들과 협력하여 제작한 웹페이지 입니다. Vue.js와 Spring을 이용하여 만들었습니다.

Kakao의 지도 API를 사용하여 지도를 표시해주고 표시된 지도에 차량 위치를 마킹하고 대여 차량을 확인하는 기능을 구현했습니다. Kakao의 API 설명이 잘 정리되어있고 깔끔한 UI를 갖고 있어서 Kakao의 API를 사용했습니다.

Vue.js로 Front-end 쪽을 구현했고 Bootstrap Vue를 이용하여 30% 정도 가량의 UI 제작에 도움을 받았습니다. 차량 대여 시간을 계산하는 방식은 Unixtime Stamp를 이용하여 시간 계산을 하였습니다.
해당 페이지를 만들면서 Vue.js에 대한 지식도 많이 얻고, API의 중요성을 알 수 있게 해준 프로젝트였습니다.
주요 기능
차량 위치 확인, 차량 가격 확인, 차량 정보 확인, 위치 검색, 대여하기, 반납하기, 관리자 유저 및 차량 관리
